Dynamic Sound
=============

  1. `init`
    * capture from cam
    * play 4 sounds
  2. `while 1`
    * query frame (from cam)
    * sub: image(t) - image(t-10) = moved
    * weight each 4 quart of the frame (0.0 < weight < 1.0)
      * moved / 4 quarter images -> sum each quarter px -> % -> volume
    * apply weight as channel volume
